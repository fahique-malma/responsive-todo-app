A responsive and interactive Todo List application built using **HTML, CSS, and JavaScript**. This app helps users organize their daily tasks efficiently with features such as adding, editing, completing, and deleting tasks.

---

## ✨ Features

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Add tasks using the Enter key
* Fully responsive design for mobile, tablet, and desktop
* Modern gradient UI with smooth animations
* Lightweight and fast performance

## ⚙️ How It Works

### Adding a Task

* Enter a task in the input field.
* Click the Add button or press Enter.
* The task is added to the task list.

### Completing a Task

* Click the circular checkbox.
* The task will be marked as completed.
* Completed tasks display a checkmark and strikethrough text.

### Editing a Task

* Click the edit icon (✏️).
* Update the task text in the prompt box.
* Save the changes.

### Deleting a Task

* Click the delete icon (❌).
* The task is instantly removed from the list.
